# AWS NGINX Portfolio Website 🚀

This project demonstrates hosting a static portfolio website using NGINX on an AWS EC2 instance.

## 🔧 Tech Stack
- AWS EC2
- NGINX
- Linux (Ubuntu)
- HTML/CSS

## 🚀 Features
- Static website hosting
- Web server configuration using NGINX
- Public access via EC2 Public IP
- Security group configuration (HTTP access)

## 🌐 Deployment Steps
1. Launch EC2 instance
2. Install NGINX
3. Deploy website files to /var/www/html
4. Configure Security Group (Port 80)
5. Access via Public IP

## 📌 Outcome
Successfully hosted a live website accessible over the internet using AWS infrastructure.
